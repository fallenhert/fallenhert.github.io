## 2024/06/07 星期五

### git 相关命令

修改/删除提交的 git 记录 - 基于 git rebase 的操作

git log: 查看 git 提交记录

git rebase -i <\hash>: 使用 git rebase 命令重定位提交记录，会自动打开编辑器

在编辑器中编辑，对于需要修改的记录、将前缀从 pick 改为 edit；对于需要删除的记录、将前缀从 pick 改为 drop；

对于需要修改的记录，改前缀为 edit 后，保存关闭编辑器，rebase 会自动 continue 并停在记录处打开编辑器。在编辑器中编辑对应修改项即可。

对于需要删除的记录，改前缀为 drop 后，保存关闭编辑器，会删去对应记录。

之后通过 git push --force 推送到远端仓库。
