## 2024/06/14 星期五

### 工作内容

[可编辑表格](https://procomponents.ant.design/components/editable-table)

研究了一下`EditableProTable`相关，浏览了一些踩坑，这边做一些记录。

`EditableProTable`自带有功能按钮，可以通过设置`recordCreatorProps={false}`关闭自带的按钮。如果想要控制新建行，可以使用`actionRef.current?.addEditRecord(row, options)`来实现。其中，row 包含行信息（核心为 key、必须唯一），options 包含其他选择信息（用到了 position 设定添加位置）。具体还请参考文档和代码。

点击编辑会开启编辑模式，编辑模式下的行属性可以通过`editable`来配置。具体的配置信息可以参考官方文档的`TableRowEditable`部分。此外可以通过`actionRender`方法来控制默认展示选项。

```js
actionRender: (row, config, defaultDoms) => {
  return [defaultDoms.save, defaultDoms.cancel];
},
```

上文代码即控制选项仅展示`保存`和`取消`两个按钮。

昨天的 Tree 组件更新了同级展开互斥的功能，通过 `onExpand`维护。

不得不感慨 antd 这边提供组件功能完整和强大。
