### 工作内容

Object 对象相关方法，可以处理 jsonObject 格式对象获取 key 值和 value 值

```js
jsonObject = { a: 1, b: 2 };

var keys = Object.keys(jsonObject); // get all keys
// keys = ["a", "b"];

var values = Object.values(jsonObject); // get all values
// values = [1, 2];

var entrys = Object.entries(jsonObject);
// entrys = [["a", 1], ["b", 2]]
```
