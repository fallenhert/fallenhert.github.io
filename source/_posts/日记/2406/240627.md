### 工作内容

今天迭代了 inputset 相关的需求。

为独立的一个表格对象添加属性，并且让它能够和其他的 div 对象靠齐。至于为什么要单独用这一个表单对象，因为需要套用 required 的红星样式。
接触到了如何在 TSX 文件中写 CSS 并且覆盖到`<Form.Item>`的方法。下面用 chatGPT 给的案例来描述。

```js
import { Form, Input } from "antd";
import React from "react";

const DemoForm: React.FC = () => {
  const formItemLayout = {
    labelCol: { style: { width: "16%" } },
    wrapperCol: { style: { width: "84%" } },
  };

  return (
    <Form>
      <Form.Item
        {...formItemLayout}
        label="Name"
        name="username"
        rules={[{ required: true, message: "Please input your username!" }]}
      >
        <Input />
      </Form.Item>
      {/* 更多 Form.Item... */}
    </Form>
  );
};

export default DemoForm;
```

接触到了下载功能的写法，好像就是 window 打开一个文件地址窗口。数据库会下发对应数据包到浏览器。
