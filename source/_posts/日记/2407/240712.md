之前接触过的妙妙小工具 MonacoEditor 需要在渲染的时候绑定长和宽。如果需要在模态框中渲染 MonacoEditor，需要在模态框展示之后、再作渲染，已保证 MonacoEditor 能够绑定上对应的长宽属性。

前人代码中留存的写法是通过 setTimeOut 发起一个宏任务，修改 MonacoEditor 绑定的 key 来进行渲染刷新。

```js
setTimeOut(() => {
  setEditorKey((prevKey) => prevKey + 1);
}, 0);
```
